数据说明：

1. 分类语料来自搜狗实验室，包含训练集和测试集两个目录。

（1）训练集
Training Dataset目录下的9个子目录，每个子目录对应一个类别，共包含9个类别的数据，每个类别包含1989篇文本。

类别代号与相应的类别对应关系如下：

C000008	财经  
C000010	IT
C000013	健康
C000014	体育
C000016	旅游
C000020	教育
C000022	招聘
C000023	文化
C000024	军事

（2）测试集
Test Dataset目录下的9个待分类的文本。

2. 分词词典
SegDict.TXT中共包含217146个词语。
